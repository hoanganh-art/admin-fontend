Demo: Hiển Thị Thông Tin Tài Khoản

Hướng dẫn trực quan cách hệ thống hiển thị thông tin người dùng đã đăng nhập

1 LocalStorage Chưa Có Dữ Liệu
Khi chưa đăng nhập, localStorage không có auth_token và employee_info
// Kiểm tra localStorage console.log(localStorage.getItem('auth_token')); // null console.log(localStorage.getItem('employee_info')); // null
Kết quả: Hệ thống redirect về trang login vì chưa có dữ liệu xác thực
2 Thêm Dữ Liệu Vào LocalStorage
Sử dụng tool test hoặc đăng nhập để thêm dữ liệu
// Thêm token localStorage.setItem('auth_token', 'test_token_123'); // Thêm thông tin nhân viên const employeeInfo = { id: 1, name: 'Nguyễn Văn Admin', email: 'admin@phonestore.com', role: 'admin' }; localStorage.setItem('employee_info', JSON.stringify(employeeInfo));
3 Hệ Thống Load Thông Tin
File user-profile.js tự động load và hiển thị thông tin
// File: js/user-profile.js function loadUserInfo() { // Lấy thông tin từ localStorage const employeeInfo = Auth.getEmployeeInfo(); if (employeeInfo) { // Cập nhật tên document.getElementById('userName').textContent = employeeInfo.name; // Cập nhật role const roleMap = { 'admin': 'Quản trị viên', 'manager': 'Quản lý', 'employee': 'Nhân viên' }; document.getElementById('userRole').textContent = roleMap[employeeInfo.role]; // Cập nhật avatar (lấy chữ cái đầu) const initials = getInitials(employeeInfo.name); document.getElementById('userAvatar').textContent = initials; } }
